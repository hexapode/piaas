π

As a Service




GET http://piaas.org/pi

Parameters

from (default : 0), the index of the digit you want to get, max 1.000.000.000.
digits (default : 1000), the number of the digit you want to get, max 100.000.
json if set return format will be JSON
jsonp if set return value will be JSONP, and the function wich name is passed will be called



Examples

GET http://piaas.org/pi?digits=16

1415926535897932

http://piaas.org/pi?digits=16&from=2015

9468398352595709

http://piaas.org/pi?digits=16&from=2015&json=1

{"num":"9468398352595709","digits":16,"from":2015}

http://piaas.org/pi?digits=16&from=2015&sonp=pi

;pi({"num":"9468398352595709","digits":16,"from":2015});
